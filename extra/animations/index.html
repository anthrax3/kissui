Another cool feature of Kissui is its animations. It has a some useful builtin CSS3 animations, thanks to Animate.css. We always keep the latest version of Aniamte.css.

It is possible to add animations to all elements of Kissui including headings, icons and even grids. To animate an element simply add animated <animation-name> class name to your target element where animation-name is the animation type you want.

Example


<h1 class="animated infinite bounce">Example</h1>

Note: You may also want to include the class infinite for an infinite loop.

Or adding animations to Icons:


<i class="fa fa-heart fa-5x animated tada infinite"></i>
