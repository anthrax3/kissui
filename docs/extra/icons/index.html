Icons

Kissui comes with tons of awesome icons, thanks to FontAwesome. Kissui has an embedded version of FontAwesome and we update it as soon as FontAwesome releases a new update.

FontAwesome has many cool icons and using them is as easy as adding fa fa-<icon> to your <i> element where <icon> is your icon name. Here is an example:

fa fa-heart
fa fa-heart fa-lg
fa fa-heart fa-2x
fa fa-heart fa-3x
fa fa-heart fa-4x
fa fa-heart fa-5x 

<i class="fa fa-heart"></i> fa fa-heart
<i class="fa fa-heart fa-lg"></i> fa fa-heart fa-lg
<i class="fa fa-heart fa-2x"></i> fa fa-heart fa-2x
<i class="fa fa-heart fa-3x"></i> fa fa-heart fa-3x
<i class="fa fa-heart fa-4x"></i> fa fa-heart fa-4x
<i class="fa fa-heart fa-5x"></i> fa fa-heart fa-5x

You can simply find and select your icon name here. It is possible to use icons with Headings and Menus.

Last but not least, you can add animated icons:
